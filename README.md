# 🌱 Agro Products Management System

A Java-based project built with **IntelliJ IDEA** and **MySQL** to manage agricultural products efficiently.  
This system provides CRUD (Create, Read, Update, Delete) operations for products and services, making it easier to track, update, and manage agro-based inventory.

---

## 🚀 Features
- Add new products with details (name, category, price, quantity).
- Update existing product information.
- Delete products from the system.
- View all products in a structured format.
- Service layer implementation (`ProductServiceImpl`) for business logic.
- Interface-driven design (`Operable.java`) for flexibility and scalability.

---

## 🛠️ Tech Stack
- **Java** (Core + OOP concepts)
- **MySQL** (Database management)
- **IntelliJ IDEA** (Development environment)
- **Git & GitHub** (Version control)
